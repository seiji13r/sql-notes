# ASCENDING AND DESCENDING ORDERING ACCORDING TO TYPE <!-- omit in toc -->

- [NUMERIC](#numeric)
- [DATE](#date)
- [STRINGS](#strings)

# NUMERIC
ASC  -> Lower on TOP
DESC  -> Higher on TOP

# DATE
ASC  -> OLDER on TOP
DESC  -> NEWER or MOST RECENT on TOP

# STRINGS
ASC  -> A on TOP
DESC  -> Z on TOP
